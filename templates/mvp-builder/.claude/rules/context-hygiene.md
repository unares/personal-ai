## Context Hygiene

- Run /compact when approaching 70% context usage
- Write HANDOFF.md before long sessions end (use /handoff)
- Keep functions < 30 lines, files < 300 lines
- Prefer editing over rewriting entire files
- Use subagents for research-heavy tasks to protect main context
- Don't dump entire file contents into conversation — read targeted sections
- Use query_context MCP tool for AIOO context (snippets only, not full vault)
